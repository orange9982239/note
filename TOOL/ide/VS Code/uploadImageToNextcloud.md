# upload image

## 效果
1. catch image
    <kbd>Win</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd>

2. upload to nextcloud and creat img link
    <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>

3. 靜待圖片上傳
    * 你剪貼簿中的圖片被上傳到nextcloud中。
    * 並寫入為markdown格式的圖片連結。
        ``` markdown
        ![](https://nextcloud.dev.nttu.edu.tw/s/NC4BJspQRj4QyRN/preview)
        ```
